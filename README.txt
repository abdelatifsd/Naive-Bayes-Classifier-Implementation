Instructions on how to the run the code:

1- Go to jupyter.org, and press on the try it in browser button.
2- In the new page, click on file in the menu bar, and then click on open.
3- In the newpage, you can find (upload) on the right side, click on it and upload the 472-AI.ipynb file.
4- Run the all cells and follow the instructions given.

PS: If you're familiar with anaconda and already have jupyter notebook installed, simply open the command line
in this directory and type the following command: jupyter notebook
After that you can open the 472-AI.ipynb file and run the cells.
